## Progressive enhancement
Progressive enhancement is een webontwikkelingsbenadering die begint met het bouwen van een basisversie van een website met eenvoudige technologieën. Vervolgens worden geavanceerdere functies toegevoegd op basis van de mogelijkheden van de gebruikte technologieën en browsers. Het doel is om een inclusieve en toegankelijke gebruikerservaring te bieden voor alle gebruikers, ongeacht de gebruikte apparaten of browsers.

## Gebruik
Aan de hand van verschillende user-stories ga ik te werk bij het maken van mijn website.

## Kenmerken
De website wordt aan de 'client-side' gebouwd aan de hand van html, css en javascript. Door midden van de server-side te implementeren in de (nu nog) statische site maak je de website dynamisch met recente data. Dit gedeelte draait op node.js en express.

## Installatie
1. Fork de repository vanuit de fdnd pagina: the-web-is-for-everyone-interactive-functionality.
2. Installeer node.js
3. Typ 'npm install' in op je terminal
4. Typ vervolgens 'npm start'
5. Nu draait de website als het goed is
